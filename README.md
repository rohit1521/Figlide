# Figma to PPT Plugin

A Figma plugin that exports designs to PowerPoint presentations.

## Development

### Prerequisites
- Figma Desktop app
- Node.js and npm

### Setup
1. Clone this repository
2. Install dependencies: `npm install`
3. Open Figma Desktop app
4. Import plugin from manifest.json
